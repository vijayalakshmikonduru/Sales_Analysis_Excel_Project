📊 Sales Data Analysis & Dashboard (Excel Project)

📌 Project Overview
This project focuses on analyzing sales data using Microsoft Excel to derive actionable business insights.
It includes data cleaning, exploratory data analysis (EDA), KPI tracking, and an interactive dashboard to visualize 
performance across different dimensions such as country, category, and device type.

🎯 Objectives
Clean and prepare raw sales data for analysis
Identify key performance indicators (KPIs)
Perform exploratory data analysis (EDA)
Build an interactive dashboard for decision-making
Analyze sales trends across regions, categories, and devices

📂 Project Structure

Total Revenue

Dashboard
Interactive visual dashboard with charts and slicers

📊 Dataset Description
The dataset includes the following key columns:
Country – Location of sales
Category – Product category (e.g., Books, Games, Clothing)
Revenue – Total sales revenue
Profit – Profit generated
Device Type – Platform used (Mobile, PC, Tablet)
Order ID – Unique identifier for each order

🔍 Key Insights
Revenue and profit vary significantly across countries
Certain product categories contribute more to overall profit
Device type influences purchasing behavior
Seasonal or regional trends can be identified through EDA
